# steelconnect_bulk_delete
Deletes all sites from SteelConnect Manager.

## Getting Started
USAGE:
    steelconnect_bulk_delete.py scm.riverbed.cc organization

### Prerequisites
Requires Python 3.

Requires the Requests library:
- pip3 install requests
